# Module `saved-posts` — Enregistrements

**Statut : TODO — implémentation prévue à l'étape 4 du Lot 1.**

Rôle : permettre à l'utilisateur d'enregistrer des posts pour les retrouver
plus tard, organisés par catégories.

Règles métier clés :
- catégorie par défaut **« Général »** créée pour chaque utilisateur ;
- au Lot 1, l'utilisateur enregistre un post dans cette catégorie
  (catégories personnalisées envisagées plus tard — TODO Lot 2+) ;
- l'enregistrement est privé (invisible pour l'auteur du post) ;
- action accessible depuis chaque post du feed, aux côtés de la réaction,
  du commentaire et du partage.
